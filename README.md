# fun
Utilities for fun-having.  

root/
- ascii_hash.py - Convert target file into ascii art in-terminal  
- funkytown.py - Script for visualization of hashing process  
- name.py - Script to convert text into cryptographic username  

misc/  
- DD1_TowerStacking_Script_Modified.ahk - AutoHotKey 1.0 script for automation of Dungeon Defenders tower stacking  
- capstoctrl.reg - Simple script to convert caps / ctrl in registry on Windows 10  
- datetime.lua - OBS Import Script for display of time
